# Logo de la Ressourcerie

Pour ajouter le logo de la Ressourcerie :

1. Placez votre fichier logo dans ce dossier (`static/images/`)
2. Nommez-le exactement : `logo-ressourcerie.png`
3. Formats acceptés : PNG, JPG, SVG (recommandé : PNG avec fond transparent)
4. Taille recommandée : largeur maximale 400px, hauteur maximale 120px

Le logo remplacera automatiquement le texte "La ressourcerie" sur la page d'accueil.

Si le logo n'est pas trouvé, le texte "La ressourcerie" s'affichera automatiquement à la place.
